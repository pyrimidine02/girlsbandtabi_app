## 🛠️ 기술 스택 개요

### 백엔드 핵심 기술
- **언어**: Kotlin 2.0.21
- **프레임워크**: Spring Boot 3.3.4
- **아키텍처**: Spring MVC + JPA
- **데이터베이스**: PostgreSQL 15 + PostGIS
- **캐시**: Caffeine (L1) + Redis (L2)
- **보안**: Spring Security + JWT
- **빌드**: Gradle 8.x

### 개발 도구 및 품질
- **코드 스타일**: ktlint + detekt (Google Style)
- **테스팅**: JUnit 5 + TestContainers
- **문서화**: OpenAPI 3.1 + Swagger UI
- **모니터링**: Micrometer + Prometheus
- **컨테이너**: Docker + Docker Compose

## 🚀 빠른 실행 가이드

### Option 1: Docker Compose (권장)
```bash
# 1. 환경 변수 설정
cp .env.example .env

# 2. 서비스 실행
docker compose up -d

# 3. 애플리케이션 확인
curl http://localhost:8080/actuator/health
```

### Option 2: 네이티브 실행
```bash
# 1. 데이터베이스 실행
docker compose up -d db redis

# 2. 애플리케이션 실행
./gradlew bootRun --args='--spring.profiles.active=local'
```

### 주요 URL들
- **API 문서**: http://localhost:8080/swagger-ui.html
- **헬스 체크**: http://localhost:8080/actuator/health
- **메트릭**: http://localhost:8080/actuator/prometheus
