# ADR-20260211: Pastel Primary Theme

## Status
Accepted

## Context
The app’s primary brand color was a saturated indigo. The desired direction is
to move to a softer pastel-toned main color while keeping contrast readable.

## Decision
- Update the primary palette to a pastel periwinkle tone.
- Adjust gradients and ripple color to align with the new primary.
- Use dark text on the light primary in the light color scheme.

## Alternatives Considered
- Keep the existing indigo and only lighten containers.
- Switch to a pastel pink as primary (conflicted with existing secondary).

## Consequences
- The overall UI feels softer and lighter.
- Primary buttons now use dark text for better contrast on pastel backgrounds.
