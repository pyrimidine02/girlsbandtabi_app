# ADR-20260308-feed-header-community-menu-scope

## Status
Accepted (2026-03-08)

## Context
- 사용자 요청: 피드 상단 삼선 버튼은 바텀시트가 아니라
  현재 설정 화면과 유사한 커뮤니티 설정 페이지로 진입해야 함.
- 기존 바텀시트 접근은 빠른 액션에는 유리하지만,
  설정 스타일 일관성과 확장성(항목 추가/권한 분기)에 불리함.

## Decision
- 피드 섹션 헤더 삼선(`Icons.menu_rounded`) 버튼은
  전용 페이지 `/community-settings`로 이동한다.
- 신규 `CommunitySettingsPage`는 기존 설정 페이지와 동일 계열 카드/리스트
  디자인을 사용한다.
- 페이지 기능 구성(프로필 우선):
  - 내 프로필
  - 팔로워
  - 팔로잉
  - 알림함
  - 저장한 글
  - 게시글 작성
  - 알림 설정
  - 계정 도구
  - 운영 센터(권한 사용자)

## Consequences
- 장점:
  - 기존 설정 UX와 디자인 일관성 확보.
  - 커뮤니티 항목 확장/권한 분기에 유리.
  - 피드 헤더 액션 의미가 명확해짐(커뮤니티 설정 진입).
- 단점:
  - 1탭 즉시 액션(바텀시트 직접 실행)보다 한 단계 진입이 추가됨.
