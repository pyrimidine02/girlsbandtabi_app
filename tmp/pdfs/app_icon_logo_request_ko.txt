앱 아이콘/로고 디자인 요청서

안녕하세요.
GirlsBandTabi 앱 리브랜딩 작업 관련하여 앱 아이콘/로고 디자인을 요청드립니다.

1. 프로젝트 개요
- 현재 프로젝트명: GirlsBandTabi
- 서비스 성격: 음악/공연/장소/커뮤니티 기반 팬 앱
- 핵심 톤앤매너: 친근함, 생동감, 팬 커뮤니티 감성, 모바일 가독성
- 참고: 추후 서비스 확장 시 앱 이름(브랜드명) 변경 가능성이 있음

2. 작업 목적
- 홈 화면에서 식별성 높은 앱 아이콘 제작
- 브랜드 인지 가능한 심볼형 로고 확립
- iOS/Android 스토어 및 앱 내 공통 사용 가능한 로고 시스템 확보
- 향후 이름 변경/서비스 확장에도 재활용 가능한 브랜드 자산 구축

3. 디자인 방향
- 키워드: 음악, 라이브, 팬덤, 커뮤니티, 확장성
- 확장 고려사항:
  - 현재 걸즈밴드 중심이지만, 향후 성우 및 개인 아티스트까지 범위 확장 가능
  - 특정 장르에 과도하게 종속된 모티프(예: 특정 밴드 악기만 강조)는 지양
  - 텍스트 없는 심볼 단독으로도 브랜드 식별 가능해야 함
- 스타일: 단순한 형태, 작은 사이즈에서도 인식 가능한 실루엣
- 컬러: 현재 앱 주요 컬러와 조화되되, 단색(모노톤)에서도 식별 가능
- 지양사항: 과도한 디테일, 텍스트 의존형 로고, 저대비 조합

4. 요청 산출물
- 앱 아이콘 시안 3안 (방향성 다른 컨셉)
- 최종 1안 확정 후 변형 세트
- 로고 심볼 단독 버전 + 심볼/워드마크 조합 버전
- 라이트/다크 배경 대응 버전
- 단색(모노크롬) 버전

5. 파일 스펙
- 원본: AI 또는 SVG (벡터)
- 프리뷰: PNG (최소 1024x1024)
- 앱 아이콘:
  - iOS용 1024x1024 마스터(비투명 배경 기준)
  - Android Adaptive Icon용 foreground/background 분리 원본
  - Android 13 monochrome 아이콘용 단색 버전
- 전달 포맷: 원본 + PNG 미리보기 + 간단 가이드(PDF 1p)

6. 사용 범위
- 앱 런처 아이콘
- 앱 스토어(Apple/Google) 등록 이미지
- 앱 스플래시/브랜딩 영역
- 향후 SNS/웹 썸네일 활용 가능 범위 포함

7. 참고 요청
- 시안별 의도 설명 2~3줄 첨부 부탁드립니다.
